# Unity Project (scaffold)

This repository contains a minimal Unity project scaffold.

Unity version: Unity 6.0 (6000.0.72f1) LTS

How to open:
1. Open Unity Hub and add this folder as a project, or open the folder directly in Unity Editor.
2. Unity will generate `Library/` and full `ProjectSettings/` on first load.

Git:
- Git LFS is configured for common asset types. Run `git lfs install` if needed.

Next steps:
- Open the project in Unity and create your first scene under `Assets/`.
- If you want, I can add a starter prototype scene and example scripts.
